About AI Medical Recommendation System

This is an AI-based medical recommendation system developed using machine learning algorithms. The system utilizes the Support Vector Classifier (SVC) algorithm for disease prediction.

The dataset used for training the SVC model is sourced from Kaggle, a platform for data science and machine learning competitions.

The system analyzes symptoms input by users and predicts the most likely disease based on those symptoms. It then provides recommendations such as description, precautions, medications, diets, and workouts related to the predicted disease.
